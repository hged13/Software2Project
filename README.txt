DESKTOP SCHEDULING APPLICATION
made to schedule appointments from several locations with several contacts and customers.
Author: Hannah Gedlaman
post 9-15-2020 application version.
6/14/2021
IDE : IntelliJ IDEA 2020.3
Java Se 11.0.9
Java FX SDK 11.0.9
Run the program. Login with a valid username and password('test' and 'test' will work), click the 'customers' button to add, update and view customers, the 'appointments' button to add, update and view appointments
and the 'reports' button to view reports and a schedule for each contact.

Additional reports include:
a total running number of appointments
a total running number of customers
the current contact(s) with the most appointments.

 mysql-connector-java-8.0.22
